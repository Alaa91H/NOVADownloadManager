# Privacy Policy

## Local Storage
All extension data, including settings, cached state, and diagnostic information, is stored exclusively on the user's device using `browser.storage.local`. No data is uploaded to external servers.

## Tokens
Authentication tokens and session credentials are handled locally and are never transmitted outside the extension. Tokens are redacted from all logs and diagnostic exports to prevent accidental exposure.

## Diagnostics
Diagnostic data is collected solely for debugging and troubleshooting purposes. This data remains on-device and is never sent to any remote endpoint. Users can inspect, export, or clear diagnostics at any time from the extension settings.

## No Telemetry
The extension does not include any telemetry, analytics, or usage tracking. There are no background communication channels, beacon calls, or third-party analytics libraries. The extension operates entirely offline with respect to user data.
